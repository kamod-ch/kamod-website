# kamod-website

PreactPress-Umsetzung der bestehenden Kamod-Website aus `../website`.

## Entwicklung

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Struktur

- `index.mdx` – Onepager-Inhalt
- `.preactpress/config.ts` – Site-Konfiguration
- `.preactpress/theme/Layout.tsx` – eigenes Layout
- `.preactpress/theme/site.css` – Styles
- `public/` – statische Assets
